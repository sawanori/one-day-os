## 4. デザイン（Brutalist）

### 4.1 カラースキーム
- [ ] 背景色が純黒（#000000）である
- [ ] 前景色が純白（#FFFFFF）である
- [ ] アクセントカラーが赤（#FF0000）である
- [ ] IH値に応じた色分けが正しく機能する
  - [ ] IH > 80%: 緑色
  - [ ] IH 50-80%: 黄色
  - [ ] IH 20-50%: オレンジ色
  - [ ] IH < 20%: 赤色

### 4.2 タイポグラフィ
- [ ] フォントが等幅フォント（Courier New）である
- [ ] フォントサイズが適切に設定されている
- [ ] 行間が適切に設定されている
- [ ] 日本語テキストが正しく表示される

### 4.3 レイアウト
- [ ] ボーダーが直角（border-radius: 0）である
- [ ] ボーダー幅が2pxである
- [ ] スペーシングが適切に設定されている
- [ ] グラデーションが使用されていない
- [ ] 柔らかいシャドウが使用されていない

### 4.4 GlitchText
- [ ] IH値に応じてグリッチエフェクトが変化する
- [ ] IH 100-81: グリッチなし
- [ ] IH 80-61: 低グリッチ
- [ ] IH 60-41: 中グリッチ
- [ ] IH 40-21: 高グリッチ
- [ ] IH 20-0: クリティカルグリッチ
